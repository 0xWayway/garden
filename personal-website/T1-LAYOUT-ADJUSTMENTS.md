## 📐 尺寸计算

### 3cm 转换为像素

标准屏幕 DPI: 96 (Windows) / 72 (Mac)

计算公式:
```
像素 = cm × DPI / 2.54

使用 96 DPI:
113px = 3cm × 96 / 2.54
```

**最终下移距离**:
- 原有顶部间距: 48px (pt-12)
- 额外下移: 113px (3cm)
- **总计**: 161px

## 🌸 花朵动画说明

**为什么花朵位置不变？**

花朵动画使用视口高度单位 (vh):
```tsx
top: ['0vh', '12vh', '12vh']
```

- 0vh = 视口顶部
- 12vh = 视口高度的 12%
- 与内容的 padding 无关

**效果**:
- 内容下移，但花朵仍从最顶部开始
- 花朵停在相对顶部 12% 的位置
- 内容下移后，花朵和内容之间有更多空间
